# 🏫 智慧校园AI向导

> 本项目是浙江树人学院《人工智能》课程大作业

---

## 📖 项目简介

智慧校园AI向导是一款基于扣子AI（Coze）平台和Trae AI编程工具开发的交互式校园服务系统。系统通过自然语言对话的方式，为浙江树人学院师生提供校园信息查询、文案内容生成和校园场景导览三大核心功能。用户无需下载安装，在浏览器中打开即可使用。

---

## ✨ 核心功能

### 1. 校园信息问答
用户可通过点击快捷按钮或输入文字，询问图书馆开放时间、食堂推荐菜品、体育馆开放时间等校园基础信息，AI基于预设知识库给出准确回答。

### 2. 校园文案生成
用户点击「新生指南」「一日游攻略」「招新文案」等按钮，AI自动生成对应的校园文案内容，适用于活动宣传、新生引导等场景。

### 3. AI校园导游 + AIGC视觉展示
用户点击「图书馆」「食堂」「教学楼」场景按钮，系统会同时切换展示AIGC生成的校园场景插画，并调用AI生成该地点的详细介绍。

## 🛠️ 技术架构

| 层级 | 技术/工具 | 说明 |
|------|----------|------|
| 前端 | HTML5 + CSS3 + JavaScript | 页面结构与交互逻辑 |
| AI服务 | 扣子AI（Coze）平台 | 智能体托管与对话API |
| 大模型 | 豆包（Doubao） | 自然语言处理 |
| AI编程辅助 | Trae AI | 代码生成与调试 |
| AIGC资产 | 通义万相 | 校园场景插画生成 |
| 版本管理 | Git + GitHub | 代码托管 |

---

## 🚀 运行方式

1. 克隆或下载本仓库到本地
2. 双击 `index.html` 文件，在浏览器中打开
3. 点击右侧功能卡片中的快捷按钮，或输入文字与AI对话

> 注意：需要互联网连接以调用扣子AI API
